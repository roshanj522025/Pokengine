# Pokengine Android App

An Android WebView wrapper app for [Pokengine](https://www.pokengine.org) — a platform to create and play your own Pokémon fan games.

## Features

- Full WebView experience of pokengine.org
- Swipe-to-refresh support
- Progress bar while loading
- Back navigation within the app
- Cookie support for login sessions
- JavaScript and DOM storage enabled for full site functionality
- External links open in the device browser

## Tech Stack

- Java (Android)
- WebView with full JS/cookie support
- SwipeRefreshLayout
- Material Components theme

## Build

Open in Android Studio and run, or:

```bash
./gradlew assembleDebug
```

Requires Android SDK 21+.
